# LAdmin
基于Vue.js, 使用Vuetify构建的后端管理系统解决方案
## 功能
- 登录/登出
- 支持拖拽的Tab导航
- 面包屑导航
- 收缩侧边栏
- 夜间模式
- 列表
   - 支持依靠函数，表头配置直接显示
   - 支持刷新，搜索，跳页控件
   - 支持自定义筛选，自定义表头
- 输入
   - 时间输入控件
   - 文件输入控件
